date对象详解【翻译自Dash javascript文档】

发表于 2016-03-29   |   分类于 javascript ， js内置对象 ， 翻译练习 ， Dash   |  

Creates a JavaScript Date instance that represents a single moment in time.
Date objects are based on a time value that is the number of milliseconds since 1 January, 1970 UTC【协调世界时】.

创建一个代表某一时刻时间的js Date对象，
Date对象时间是相对于1970年7月1日（世界统一时）的毫秒数

Syntax [‘sɪntæks]【语法】

  • Note: JavaScript Date objects can only be instantiated by calling JavaScript Date as a constructor: calling it as a regular function (i.e. without the new operator) will return a string rather than a Date object; unlike other JavaScript object types, JavaScript Date objects have no literal syntax.

  • 注意：JS Date对象实例只能以Date对象作为构造函数创建：如果将Date当做一个普通的函数（也就是没有new关键字）则会返回一个字符串而不是一个Date对象；与其他JS对象不同的是，Date对象没有字面量语法

Parameters [pə’ræmətɚ]【参数】

  • Note: Where Date is called as a constructor with more than one argument, if values are greater than their logical range (e.g. 13 is provided as the month value or 70 for the minute value), the adjacent value will be adjusted. E.g. new Date(2013, 13, 1) is equivalent to new Date(2014, 1, 1), both create a date for 2014-02-01 (note that the month is 0-based). Similarly for other values: new Date(2013, 2, 1, 0, 70) is equivalent to new Date(2013, 2, 1, 1, 10) which both create a date for 2013-03-01T01:10:00.

  • 注意： 当Date对象作为构造函数被传入多个参数，若传入的值超出合理的范围（比如 月份中传入13或者分钟中传入70）,则上一级时间的值会被调整。例如 new Date(2013, 13, 1)相当于 new Date(2014, 1, 1)，都是创建了一个指向2014年2月1号的日期对象（注意月份是从0开始的）。类似地，new Date(2013, 2, 1, 0, 70)和Date(2013, 2, 1, 1, 10)一样都创建了一个date对象指向2013年3月1号1点10分

1.Value 基本数值

  • Integer value representing the number of milliseconds since 1 January 1970 00:00:00 UTC (Unix Epoch).

  • 一个相对于1970年1月1日0时的毫秒数（整数）

2.dateString 时间字符串

  • String value representing a date. The string should be in a format recognized by the Date.parse() method (IETF-compliant RFC 2822 timestamps and also a version of ISO8601).

  • 一个可以被Date.parse()方法解析的时间字符串（要符合原文后面那一串标准）

3.year

  • Integer value representing the year. Values from 0 to 99 map to the years 1900 to 1999.

  • 一个代表年份的整数，0-99对应1900年到1999年

4.month

  • Integer value representing the month, beginning with 0 for January to 11 for December.

  • 一个代表月份的整数，取值范围为0（1月份）至11（12月）

5.day

  • Optional. Integer value representing the day of the month.

  • 可选项，一个代表日期的整数

6.hour

  • Optional. Integer value representing the hour of the day.

  • 可选项，一个代表小时数的整数

7.minute

  • Optional. Integer value representing the minute segment of a time.

  • 可选项，一个代表分钟数的整数

8.second

  • Optional. Integer value representing the second segment of a time.

  • 可选项，一个代表秒数的整数

9.millisecond

  • Optional. Integer value representing the millisecond segment of a time.

  • 可选项，一个代表毫秒数的整数

Description 【描述】

  • If no arguments are provided, the constructor creates a JavaScript Date object for the current date and time according to system settings.

  • 如果不传入任何参数,则创建一个date对象指向当前系统时间

  • If at least two arguments are supplied, missing arguments are either set to 1 (if day is missing) or 0 for all others.

  • 如果传入了至少两个参数，则缺少的参数会被默认设置为1（如果缺少日期）或 0（除了日期的其他参数）

  • The JavaScript date is based on a time value that is milliseconds since midnight 01 January, 1970 UTC. A day holds 86,400,000 milliseconds. The JavaScript Date object range is -100,000,000 days to 100,000,000 days relative to 01 January, 1970 UTC.

  • js date对象时间相对于1970年1月1号半夜0时，一天有八千六百四十万毫秒，js date对象日期范围为1970年1月1号0点前后1亿天

  • The JavaScript Date object provides uniform behavior across platforms. The time value can be passed between systems to represent the same moment in time and if used to create a local date object, will reflect the local equivalent of the time.

  • js date对象对跨平台提供统一的处理方式，你可以提供不同系统的时间表示方式去表示同一个时间，同时如果你在执行环境中创建了一个Date对象，这个Date对象将指向执行环节的当前时间（不知道对不对，这段看不太懂）

  • The JavaScript Date object supports a number of UTC (universal) methods, as well as local time methods. UTC, also known as Greenwich Mean Time (GMT), refers to the time as set by the World Time Standard. The local time is the time known to the computer where JavaScript is executed.

  • js Date对象提供了一系列UTC（世界协调时、世界统一时间）方法和本地时间方法。UTC（世界统一时），也可以理解为格林威治标准时间，提供全球时间参考标准。本地时间指的是js执行环境的机器时间

  • Invoking JavaScript Date as a function (i.e., without the new operator) will return a string representing the current date and time.

  • 若将Date对象当做一个普通的函数执行（没有new操作符）将会返回一个当前日期和时间的字符串

Properties 【属性】

  • Date.prototype

    Allows the addition of properties to a JavaScript Date object.
可以向js date对象添加额外属性

  • Date.length

    The value of Date.length is 7. This is the number of arguments handled by the constructor.
Date.length的值为7，这个值指的是date处理的参数的个数

Methods【私有方法】

  • Date.now()

Returns the numeric value corresponding to the current time - the number of milliseconds elapsed since 1 January 1970 00:00:00 UTC.

返回自1970年1月1号0时（世界统一时）到现在为止的毫秒数

  • Date.parse()

Parses a string representation of a date and returns the number of milliseconds since 1 January, 1970, 00:00:00, UTC.

解析一个表示日期的字符串并返回该日期距1970年1月1号0时（世界统一时）的毫秒数

  • Date.UTC()

Accepts the same parameters as the longest form of the constructor (i.e. 2 to 7) and returns the number of milliseconds since 1 January, 1970, 00:00:00 UTC.

接受与构造函数一样的参数（2至7个）并返回该日期距1970年1月1日0点的毫秒数

JavaScript Date instances【js Date对象实例】

All Date instances inherit from Date.prototype. The prototype object of the Date constructor can be modified to affect all Date instances.

所有Date实例都继承自Date.peototype(所有实例共享)，修改Date对象的原型对象上的方法和属性可以应用到所有Date实例上

Date.prototype Methods 【定义在Date.prototype的方法】

Getter(获取)

  • Date.prototype.getDate()

Returns the day of the month (1-31) for the specified date according to local time.

返回指定时间相对于当地时间的当前日期（1-31）

  • Date.prototype.getDay()

Returns the day of the week (0-6) for the specified date according to local time.

返回指定时间相对于当地时间的当前星期（0-6）

  • Date.prototype.getFullYear()

Returns the year (4 digits for 4-digit years) of the specified date according to local time.

返回指定时间相对于当地时间的四位数年份

  • Date.prototype.getHours()

Returns the hour (0-23) in the specified date according to local time.

返回指定时间相对于当地时间的小时数

  • Date.prototype.getMilliseconds()

Returns the milliseconds (0-999) in the specified date according to local time.

返回指定时间相对于当地时间当地时间的毫秒数（0-999）

  • Date.prototype.getMinutes()

Returns the minutes (0-59) in the specified date according to local time.

返回指定时间相对于当地时间的分钟数（0-59）

  • Date.prototype.getMonth()

Returns the month (0-11) in the specified date according to local time.

返回指定时间相对于当地时间的月份（0-11）

  • Date.prototype.getSeconds()

Returns the seconds (0-59) in the specified date according to local time.

返回指定时间相对于当地时间的秒数（0-59）

  • Date.prototype.getTime()

Returns the numeric value of the specified date as the number of milliseconds since January 1, 1970, 00:00:00 UTC (negative for prior times).

返回指定时间距1970年1月1日0点的毫秒数

  • Date.prototype.getTimezoneOffset()

Returns the time-zone offset in minutes for the current locale.

返回用分钟数表示的当前时间当前时区与世界统一时的时差

  • Date.prototype.getUTCDate()

Returns the day (date) of the month (1-31) in the specified date according to universal time.

  • Date.prototype.getUTCDay()

Returns the day of the week (0-6) in the specified date according to universal time.

  • Date.prototype.getUTCFullYear()

Returns the year (4 digits for 4-digit years) in the specified date according to universal time.

  • Date.prototype.getUTCHours()

Returns the hours (0-23) in the specified date according to universal time.

  • Date.prototype.getUTCMilliseconds()

Returns the milliseconds (0-999) in the specified date according to universal time.

  • Date.prototype.getUTCMinutes()

Returns the minutes (0-59) in the specified date according to universal time.

  • Date.prototype.getUTCMonth()

Returns the month (0-11) in the specified date according to universal time.

  • Date.prototype.getUTCSeconds()

Returns the seconds (0-59) in the specified date according to universal time.

  • Date.prototype.getYear()

Returns the year (usually 2-3 digits) in the specified date according to local time. Use getFullYear() instead.

返回指定日期相对于当地时间年份（通常返回2-3位数），推荐以getFullYear()方法取代这个方法。

Setter(设置)

  • Date.prototype.setDate()

Sets the day of the month for a specified date according to local time.

指定一个基于当地时间的日期

  • Date.prototype.setFullYear()

Sets the full year (e.g. 4 digits for 4-digit years) for a specified date according to local time.

指定一个基于当地时间的四位数年份

  • Date.prototype.setHours()

Sets the hours for a specified date according to local time.

指定一个基于当地时间的小时数

  • Date.prototype.setMilliseconds()

Sets the milliseconds for a specified date according to local time.

指定一个基于当地时间的毫秒数

  • Date.prototype.setMinutes()

Sets the minutes for a specified date according to local time.

指定一个基于当地时间的分钟数

  • Date.prototype.setMonth()

Sets the month for a specified date according to local time.

指定一个基于当地时间的月份

  • Date.prototype.setSeconds()

Sets the seconds for a specified date according to local time.

指定一个基于当地时间的秒数

  • Date.prototype.setTime()

Sets the Date object to the time represented by a number of milliseconds since January 1, 1970, 00:00:00 UTC, allowing for negative numbers for times prior.

指定一个距1970年1月1日0点的毫秒数，可以为负值

  • Date.prototype.setUTCDate()

Sets the day of the month for a specified date according to universal time.

指定一个基于世界统一时的日期

  • Date.prototype.setUTCFullYear()

Sets the full year (e.g. 4 digits for 4-digit years) for a specified date according to universal time.

指定一个基于世界统一时的4位数年份

  • Date.prototype.setUTCHours()

Sets the hour for a specified date according to universal time.

指定一个基于世界统一时的小时数

  • Date.prototype.setUTCMilliseconds()

Sets the milliseconds for a specified date according to universal time.

指定一个基于世界统一时的毫秒数

  • Date.prototype.setUTCMinutes()

Sets the minutes for a specified date according to universal time.

指定一个基于世界统一时的分钟数

  • Date.prototype.setUTCMonth()

Sets the month for a specified date according to universal time.

指定一个基于世界统一时的月份

  • Date.prototype.setUTCSeconds()

Sets the seconds for a specified date according to universal time.

指定一个基于世界统一时的秒数

  • Date.prototype.setYear()

Sets the year (usually 2-3 digits) for a specified date according to local time. Use setFullYear() instead.

指定一个基于世界统一时的2至3位数年份

Conversion getter(格式化方法)

  • Date.prototype.toDateString()

Returns the “date” portion of the Date as a human-readable string.

将日期转换为人类习惯字符串方式

  • Date.prototype.toISOString()

Converts a date to a string following the ISO 8601 Extended Format.

  • Date.prototype.toJSON()

Returns a string representing the Date using toISOString(). Intended for use by
JSON.stringify().

  • Date.prototype.toGMTString()

Returns a string representing the Date based on the GMT (UT) time zone. Use toUTCString() instead.

  • Date.prototype.toLocaleDateString()

Returns a string with a locality sensitive representation of the date portion of this date based on system settings.

转换为基于系统的时间字符串

  • Date.prototype.toLocaleFormat()

Converts a date to a string, using a format string.

  • Date.prototype.toLocaleString()

Returns a string with a locality sensitive representation of this date. Overrides the Object.prototype.toLocaleString() method.

  • Date.prototype.toLocaleTimeString()

Returns a string with a locality sensitive representation of the time portion of this date based on system settings.

  • Date.prototype.toSource()

Returns a string representing the source for an equivalent Date object; you can use this value to create a new object. Overrides the Object.prototype.toSource() method.

  • Date.prototype.toString()

Returns a string representing the specified Date object. Overrides the Object.prototype.toString() method.

  • Date.prototype.toTimeString()

Returns the “time” portion of the Date as a human-readable string.

  • Date.prototype.toUTCString()

Converts a date to a string using the UTC timezone.

  • Date.prototype.valueOf()

Returns the primitive value of a Date object. Overrides the Object.prototype.valueOf() method.
