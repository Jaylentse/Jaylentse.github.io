浮动元素外边距表现详解

发表于 2016-03-28   |   分类于 css   |  

注意：1.浮动元素外边距不合并 （水平垂直都不合并）【创建了一个新的BFC】
2.浮动元素外边距对正常流元素无影响

1.垂直外边距

  1.1 margin-top【影响自身位置，同时带动垂直位置下面的浮动元素】

    1.1.1 设置为正值

    1.1.2 设置为负值



  1.2 margin-bottom【影响垂直方向下面元素的位置,对自己位置无影响】

    1.1.1 设置为正值

    1.1.1 设置为负值


2.水平边距（以左浮动为例，右浮动同理）【左浮动的margin-left对应右浮动的margin-right】

  2.1 元素左浮动，设置margin-left

    2.1.1 设置margin-left正值

    2.1.1 设置margin-left负值


  2.1 元素左浮动，设置margin-right

    2.1.1 设置margin-right正值

    2.1.1 设置margin-right负值


总结

  1. margin-top影响自身位置，设置负值会把自己往上拉
  2. margin-bottom只影响元素下面的元素而不影响自身位置，正值往下推，负值往上拉（会使下面的元素覆盖自己），自身位置不变，垂直边距不合并
  3. 对于左浮动的元素margin-left影响自身，设置负值会把自己往左边拉
  4. 对于左浮动的元素margin-right会影响在其右边左浮动元素，对自身位置无影响，正值往右推，负值往左拉（会使右边的元素覆盖自己）
  5. 对于右浮动的元素margin-right影响自身，设置负值会把自己往右边拉
  6. 对于右浮动的元素margin-left会影响在其左边右浮动元素，对自身位置无影响，正值往左推，负值往右拉（会使左边的元素覆盖自己）
  7. 自身设置可以影响自身的负外边距可以使自己从父元素盒子里跑出去，而设置可以影响兄弟元素的负边距无法使兄弟元素离开父元素盒子
  8. 水平外边距过大会把在同一行的浮动元素挤到下一行
