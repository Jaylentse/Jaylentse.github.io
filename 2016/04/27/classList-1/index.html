classList（摘自官方文档）

发表于 2016-04-27   |   分类于 javascript ， 翻译   |  

The Element.classList is a read-only property which returns live DOMTokenList collection of the class attributes of the element.

Using classList is a convenient alternative to accessing an element’s list of classes as a space-delimited string via element.className.

语法

var elementClasses = elementNodeReference.classList;

elementClasses在这里是一个表示相关元素标签class属性DOMTokenList对象，如果class属性为空或者没有设置，elementClasses。length将会返回0。虽然我们可以通过add()方法和remove()方法改变classList的值，但classList属性确实是只读的

方法

  • add( String [, String] )

添加指定类名，如果有已经存在的类名，则该类名会被忽略

  • remove( String [,String] )

移除指定类名

  • item ( Number )

返回集合中指定索引值的类名

  • toggle ( String [, force] )

只传一个参数：切换该类名，如果改类名存在就删除它，如果不存在则添加它

如果传递第二个参数：若为true，则添加第一个参数为类名，若为false,则将第一个参数从类名列表中删除。

  • contains( String )

检查指定的类名是否已经存在
